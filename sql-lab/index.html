🔁 The Connection Lifecycle & Sample Data

    

      
1 · Connect
Open a network connection to the DB server using host, port, credentials.

      
2 · Create statement / cursor
Ask the connection for an object that can run SQL.

      
3 · Execute
Send your SQL string to the server; it parses, plans and runs it.

      
4 · Fetch
Read the returned rows into your program as a result set.

      
5 · Close
Release the connection so the server can free resources.
